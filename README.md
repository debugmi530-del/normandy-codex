# Normandy Codex

Полностью офлайн ИИ-ассистент для Windows: пишет тексты, объясняет код и материалы,
исправляет ошибки в коде и готовит ответы на присланные сообщения — всё в одном чате,
без переключения режимов. Инференс модели выполняется локально на компьютере
пользователя, без обращений в интернет (кроме единоразового скачивания самой модели).

## Возможности

- Единый чат — модель сама определяет, что нужно: написать, объяснить, исправить код
  или ответить на текст.
- Перетаскивание файлов и папок прямо в чат для анализа/исправления.
- Несколько профилей с раздельной историей переписки.
- Индикатор загрузки CPU/RAM в реальном времени.
- Выбор модели при первом запуске (по умолчанию Qwen2.5-Coder-7B-Instruct,
  есть более лёгкие варианты), докачка прерванной загрузки с места обрыва.
- Полное удаление всех данных приложения при деинсталляции (или вручную из настроек).
- Ускорение на GPU (CUDA/Vulkan) через `node-llama-cpp`, с автоматическим откатом на CPU.

## Разработка

```bash
npm install
npm run dev
```

Запустит Vite dev-сервер и Electron с горячей перезагрузкой окна.

## Сборка установщика Windows

```bash
npm run build
```

Соберёт renderer (Vite) и упакует всё в один `.exe`-инсталлятор через
`electron-builder` (папка `release/`). Всё, кроме самой модели ИИ, зашивается
в установщик — никаких дополнительных загрузок при установке не требуется.

## Автосборка и публикация релиза (GitHub Actions)

При пуше тега вида `v1.0.0` workflow `.github/workflows/build.yml` соберёт
Windows-инсталлятор на `windows-latest` и опубликует `.exe` в GitHub Releases
автоматически.

```bash
git tag v1.0.0
git push origin v1.0.0
```

## Стек

- Electron + React + TypeScript + Vite
- `node-llama-cpp` — локальный инференс GGUF-моделей (CPU/GPU)
- `electron-store` — хранение профилей и истории
- `systeminformation` — мониторинг CPU/RAM
- `electron-builder` (NSIS) — установщик с `deleteAppDataOnUninstall`
